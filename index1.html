AWS EC2 Storage Configuration

Hands-on Practice with Disk Management in EC2 Windows Instance

Project Overview

In this project, I connected to an AWS EC2 Windows instance using Remote Desktop and configured storage volumes. I explored disk management to verify the system drive and additional storage volumes attached to the instance.

Storage Details

  • C: Drive – 30 GB (System Drive)
  • D: Drive – 25 GB (Additional Storage Volume)
